# Code of Conduct

Hermes Agent Team is an open source project. Participation should be practical,
respectful, and focused on improving the software.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while asking for clear evidence.
- Keep technical disagreement focused on facts, tradeoffs, and reproducible behavior.
- Respect privacy and do not post secrets, personal data, or private workspace content.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information without permission.
- Repeatedly derailing technical discussion.
- Sharing exploit details publicly after being asked to use a private security path.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct.
Repeated or severe violations may result in being blocked from participating in the project.
